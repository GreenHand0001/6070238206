# 项目简介

## 核心功能
- 基于 YOLOv8 的传送带缺陷检测系统
- 支持 4 类缺陷检测：block、crack、foreign、hole
- 提供 2200 张标注数据集
- 包含训练好的模型及训练结果
- 支持图片、视频及摄像头进行检测
- 支持选择模型和自定义置信度、IOU 阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 传送带生产线的质量检测
- 工业设备维护与监控
- 自动化缺陷识别系统
- 生产过程质量控制

## 技术特点
- 使用 Python 3.9 开发
- 依赖环境：opencv-python、PyQt5、ultralytics、torch1.9
- 提供 UI 界面操作
- 支持 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
- 文件包含完整程序、UI 源文件、图标、数据集图片及运行教程
